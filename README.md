# المحاسب الذكي V3

تطبيق محاسبي أندرويد مبني بـ Flutter ويعمل محليًا داخل الهاتف، مع بناء APK عبر GitHub Actions بدون الحاجة إلى Android Studio على جهازك.

## الأقسام
- الرئيسية
- الحسابات
- سند قبض
- سند صرف
- القيود
- التقارير
- الإعدادات

## الموجود في هذه النسخة
- تصميم موبايل ناعم وألوان هادئة للعين.
- قوائم بأيقونات.
- شجرة حسابات 3 مستويات.
- فتح حساب جديد برقم تلقائي حسب نوع الحساب والحساب الأب.
- تعديل اسم الحساب وإيقافه.
- حذف آمن للحسابات: يمنع حذف الحساب إذا كان له حركات أو حسابات فرعية.
- إعدادات الشركة والسنة المالية.
- ترقيم تلقائي لسندات القبض والصرف والقيود حسب السنة المالية.
- سند قبض يولد قيدًا محاسبيًا تلقائيًا.
- سند صرف يولد قيدًا محاسبيًا تلقائيًا.
- رقم شيك تلقائي عند اختيار بنك.
- طباعة/مشاركة سند قبض وصرف PDF بعد الحفظ.
- قيود يومية مع منع حفظ القيد غير المتوازن.
- كشف حساب PDF مع فلتر تاريخ.
- ميزان مراجعة PDF مع فلتر تاريخ وإجماليات.
- دفتر اليومية PDF.
- تقرير الصندوق والبنك PDF.
- تصدير Excel للحسابات والقيود والملخص.
- نسخ احتياطي JSON ومشاركة الملف.
- استيراد نسخة احتياطية عبر لصق محتوى ملف JSON.
- حفظ محلي داخل الهاتف باستخدام SharedPreferences.

## البناء عبر GitHub Actions
ارفع الملفات إلى GitHub ثم شغل Workflow باسم:

`Build Android APK`

بعد نجاح البناء ستجد ملف APK داخل Artifacts باسم:

`accounting-pro-smart-mobile-v3-apk`

## ملاحظات مهمة
- المشروع أندرويد فقط.
- لا يحتاج Android Studio على جهازك.
- ملف ZIP لا يحتوي على مجلد Android؛ الـ Workflow ينشئ ملفات Android تلقائيًا قبل البناء.
- تقارير PDF العربية تعتمد على خطوط Google عبر مكتبة printing/pdf، لذلك قد تحتاج أول مرة إلى اتصال عند توليد PDF حتى يتم تجهيز الخط.
